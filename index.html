Project Nebula

Unity, C#, Networking

Project Nebula is a 2 player co-op mobile game which can be played over the local WiFi network. It was a team project built in Unity. I worked on designing the architecture for the game, the touch and gyroscope controls, a scalable damage system and also used Forge Networking, which is an open source networking plugin which can be used with Unity. Created custom shaders using Shader Graphs. Workflow followed AGILE SCRUM methodology and used JIRA to keep track of tasks and GITHub to maintain the project repository.
